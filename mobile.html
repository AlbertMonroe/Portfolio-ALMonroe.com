TIME-LINE APP

Title

Mobile App / UX Designer

Project

Create an appointments application design with original unique time-line and calendar selection user interface.

Solution

Design an opening screen that shows a list of upcoming appointments with easy to read graphical representations of time left until event(s). This eliminates the need to read text/details when time bars are full making non-urgent events obvious. Details for any event/appointment can be easily viewed when desired by tapping on its title.

Design unique calendar/time selector utilizing sliders providing fast paced event entries.

Tap and Swipe-able screen elements are placed, sized, and colored for easy access. Interface navigation labels are defined with minimal text for optimal usability.

Target User

Users that need to keep track of meetings and events.

Technologies / Skills

  • UI / UX Design
  • Prototyping
  • Wire-frames
  • Hi-Fidelity Mockups
  • PhotoShop

My Process / What I Did

  • Define and sketch all user task flows.
  • Create task flow diagrams with all potential user scenario task flows.
  • Create user interface and navigation that support all user task flows with usability best practices.
  • Create Hi-Fidelity mockups in Adobe Create Suite.
