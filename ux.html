UX - A/B COMPARIONS

Client

Major corporation contracted through Insight Global.

Title

UX Designer

Project

Improve the corporate intranet website for better customer usability and increased user response time.

Solution

Examine major global intranet portal and prepare a report that shows issues found with recommendations for improvement. Show report findings in a Power Point presentation along with A / B Hi-Definition Mock-up Comparisons.

Target Users

Major global retailers

Technologies / Skills

  • UI / UX Design
  • Prototyping
  • Hi-Fidelity Mockups
  • PhotoShop

My Process / What I Did

  • Examined corporate UX Style-Guide for compliance when making recommendations / mock-ups.
  • Analyzed portal site design and usability with navigation and content structure
  • Used the following four step process:
  • STEP #1:
    Evaluate website for usability issues that could hinder user response times.
  • STEP #2:
    Make recommendations to resolve issue(s) found. (For Stakeholders)
  • STEP #3:
    Display Hi-Fidelity mockup to display potential solution.
  • STEP #4:
    Make recommendations with guidelines. (For Development Team).
  • Prepare a recommendations report and give a Power-Point presenation to show findings and answer questions.

The following four slides is just one example set taken from the report to demonstrate the recommendations process. This process is repeated for each issue discovered with recommended improvements.
